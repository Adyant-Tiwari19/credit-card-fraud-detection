# Credit Card Fraud Detection

## Overview
Neural Network model to detect credit card fraud with 84% recall and 74% precision on highly imbalanced dataset

## Results
- Recall : 0.84
- Precision : 0.74
- Optimized Threshold : 0.999

## Tech Stack
- **Language:** Python
- **Platform :** Google Colab
- **Deep Learning :** Tensorflow , Keras
- **Data Processing :** Pandas , Numpy
- **Evaluation :** Scikit-learn
